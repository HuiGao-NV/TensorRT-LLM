Three Hosting Options:
1. Google Apps Script (simplest)
2. HTTP Endpoint via Cloud Run / Cloud Functions (production)

- Return JSON with text to reply
- Deploy as Google Chat App add-on
- Configure in Google Cloud Console → Chat API

Option 2 - Cloud Functions (Python):
- Flask-based HTTP handler
- Handle MESSAGE and ADDED_TO_SPACE event types
- Deploy via gcloud functions deploy
- Configure Chat API with HTTP endpoint URL

Async/Proactive Messages:
- Use google.oauth2 service account credentials
- Build chat service via googleapiclient
- spaces().messages().create() for async replies
- Support threaded replies via thread.name

Interactive Cards:
- cardsV2 with header, sections, widgets
- buttonList for actions
- CARD_CLICKED event type for button handlers

Setup Checklist:
1. Google Cloud Project with billing
2. Enable Google Chat API
3. Service Account for async messaging
4. Configure Chat API (name, connection, permissions)
5. Publish internally or to Marketplace
